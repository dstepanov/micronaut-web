Micronaut Framework 4.0.6 Released!

The Micronaut Foundation is excited to announce the release of Micronaut framework 4.0.6!

It contains patch releases of Micronaut Core and it upgrades Micronaut Spring, Micronaut JAX-RS, Micronaut Servlet, Micronaut Validation, Micronaut Redis, Micronaut Tracing, Micronaut AWS, and Micronaut Kafka modules to patch releases.

Moreover, update your application to version 4.0.3 of the Micronaut Gradle Plugins if you use Gradle.

NEXT STEPS

If you haven’t yet updated to Micronaut framework 4, this is an excellent opportunity to do so!

Please feel free to reach out to us if you need any assistance.
