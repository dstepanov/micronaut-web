Ready to start building lightning fast, lightweight microservices and serverless applications with Micronaut? Here’s everything you need to launch your next project!

The easiest way to create a new project with the latest version of Micronaut is with [Micronaut Launch](https://micronaut.io/launch/).  Just select the type of application you’d like to create along with the language, build system, test framework, and features you’d like to include.

You can download a new project with the provided configuration by selecting “Generate Project.”
