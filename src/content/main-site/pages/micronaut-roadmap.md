# Micronaut ROADMAP

We use [GitHub projects](https://github.com/orgs/micronaut-projects/projects) to organize work.

We work in [semantic versioning](https://semver.org/) phases. Continuously, we have several projects open:

- A patch GitHub Project – bug fixes / build / documentation/patch dependency updates
- A minor GitHub Project – [features / minor/major dependency updates](https://github.com/micronaut-projects/micronaut-core/wiki/Module-Upgrade-Strategy)
- A major GitHub Project for the next major version of the framework

We also keep a [RoadMap top-level project](https://github.com/orgs/micronaut-projects/projects/69), which we use to track features prioritized into minor releases

We also have a [GitHub Discussion Category](https://github.com/micronaut-projects/micronaut-core/discussions/categories/planning) to plan releases.
