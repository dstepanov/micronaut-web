# AgoraPulse Micronaut Journey

[![AgoraPulse](/micronaut-assets/main-site/wp-content/uploads/2025/01/Agorapulse-logo-blue.svg)](https://www.agorapulse.com)

> “Our journey with Micronaut started in 2018 when it was the only framework lightweight enough to run in AWS Lambda serverless environment. We trusted the team behind the framework as they already created the Grails framework we were using to build our applications. We expected they would provide the same amazing developer experience and they would be able to build the same open and friendly community that Grails have.”

*– Vladimír Oraný*

## AgoraPulse OSS Hub

Agorapulse believes in giving back to the community. They publish and maintain many [open-source Micronaut libraries](https://agorapulse.github.io/agorapulse-oss/#_micronaut_libraries).

## Combining Serverless and Backend architectures

Agorapulse technology stack combines more than 100 serverless functions and more than 50 server applications written in Micronaut Framework.

## From Grails to Micronaut

AgoraPulse transitioned a massive codebase from Grails to Micronaut over a multi-year period. They published a [blog describing their Micronaut journey](https://medium.com/agorapulse-stories/scaling-up-into-the-cloud-agorapulse-micronaut-journey-c721a1018933). They [published a series of blog posts about moving from Grails to Micronaut](https://medium.com/agorapulse-stories/goodbye-grails-hello-micronaut-0-introduction-ff7470cecf9d).
