In 2017, the Grails team at [Object Computing](https://objectcomputing.com/) set out to build an intelligent compiler that precomputed framework infrastructure to achieve the same productivity benefits gained from Spring® without the runtime overhead. Today, we celebrate 3 years from the date of the first GA release of the Micronaut framework!

Over the last 3 years, the Micronaut framework has significantly evolved how developers view startup time and memory consumption in building microservices and serverless applications. Our team has successfully demonstrated that the future of technology will be built with intelligent compilers and smaller, faster runtimes.

Since inception, the Micronaut framework has enjoyed massive enthusiasm, engagement, and adoption within the JVM community around the globe. For example, Micronaut core has 5k+ stars on GitHub and more than 300 contributors. The Framework is being used in production at Target, Minecraft, SmartThings, Stainless.ai, Skylo, and many other organizations.

Notable milestones we’ve achieved throughout the Framework’s first 3 years include:

- [**Micronaut Data**](https://micronaut-projects.github.io/micronaut-data/latest/guide/index.html). Micronaut Data™ is a productivity-boosting open source technology that dramatically improves runtime and memory performance of data access repository logic for the microservice and serverless era.
- [**Micronaut Launch**](https://micronaut.io/launch/). Micronaut Launch™ is an implementation of the Micronaut command line interface (CLI) and an online tool for creating Micronaut applications.
- [**Micronaut Foundation**](https://micronaut.io/foundation/). The Micronaut Foundation™ is a not-for-profit organization that exists to support and collectively lead the open source Micronaut project.

Without the support and contributions of our amazing community, we would not be where we are today.  The Micronaut framework stands on a firm foundation built with your pull requests, feature requests, reported issues, documentation updates, and questions answered.  We believe in the success of the Micronaut framework, and it is clear that you do too.  Thank you for everything that you do.

We are so very pleased to celebrate the technical advancements and initiatives we all have accomplished over the past 3 years. We can’t wait to see the amazing progress yet to come!

— Jen Wiese, Micronaut Foundation Community Engagement Manager
