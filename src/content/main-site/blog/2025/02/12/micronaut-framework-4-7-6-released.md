The Micronaut Foundation is excited to announce the release of Micronaut framework 4.7.6!

It contains patch releases of [Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.7.14) and [Oracle Cloud](https://github.com/micronaut-projects/micronaut-oracle-cloud/releases/tag/v4.3.8). This release upgrades to [Netty 4.1.118](https://netty.io/news/2025/02/10/4-1-118-Final.html), a patch release that fixes a critical CVE.

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
