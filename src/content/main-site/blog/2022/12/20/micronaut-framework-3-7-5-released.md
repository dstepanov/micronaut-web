The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.7.5!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.7.5)

This is a patch release, and it contains bug fixes. Moreover, update your application to version 3.6.7 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/search?term=micronaut)

If you have not yet updated to [Micronaut framework 3.7](https://micronaut.io/2022/09/21/micronaut-framework-3-7-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.5398010d44579cd3c326ff28c5086545.1660912926021.1664378508537.1664433503738.18&__hssc=207490764.2.1664433503738&__hsfp=2715702327) if you need any assistance.
