The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.5.4](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.5.4)!

This release contains bug fixes and patch releases of several modules, including [Micronaut Security 3.6.3](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.6.3), [Micronaut AWS 3.5.3](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.5.3), [Micronaut RxJava 2](https://github.com/micronaut-projects/micronaut-rxjava2/releases/tag/v1.2.2), [Micronaut GCP 4.2.1](https://github.com/micronaut-projects/micronaut-gcp/releases/tag/v4.2.1), and [Micronaut Reactor 2.2.3](https://github.com/micronaut-projects/micronaut-reactor/releases/tag/v2.2.3)

If you have not yet updated to Micronaut framework 3.5.0, this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support) if you need any assistance.
