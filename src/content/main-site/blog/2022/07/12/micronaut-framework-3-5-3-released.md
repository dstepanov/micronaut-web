The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.5.3](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.5.3)!

This release contains documentation improvements and [a bug fix that affected Grails framework 5.2.0 applications using the Micronaut framework](https://github.com/grails/grails-core/issues/12589).

If you have not yet updated to Micronaut framework 3.5.0, this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support) if you need any assistance.
