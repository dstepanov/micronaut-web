The Micronaut Foundation™ is truly appreciative of the tremendous support we receive from our sponsors. The generous financial contributions and other open source contributions from our Corporate Sponsors are essential to the continued evolution of the Micronaut® framework. Our Sponsorship program recognizes the companies whose contributions enable us to maintain and expand both the open source technologies and the assets we rely on to keep our communities informed and engaged.

Today, we are pleased to announce that [MicroStream](https://microstream.one/) has joined the Micronaut Foundation as a Silver-level corporate sponsor!

In 2013, MicroStream set out to develop a fundamentally different Java-native persistence toolset. They started with a mission to simplify the entire Java database development process and to radically accelerate runtime data processing using only Java concepts. Today, this revolutionary idea is a reality – a stable, proven solution that helps Java developers around the world build innovative new features and products. MicroStream is currently focusing on growing an open, enthusiastic, collaborative worldwide community that uses their software to turn other innovators’ visions into reality.

As part of their sponsorship and in support of our organizations’ shared values and goals within the open source ecosystem, MicroStream and the Micronaut Foundation collaborated on the development of an open source MicroStream module for the Micronaut framework. The MicroStream module is available as a preview feature in Micronaut Framework 3.5.0. You can also select several MicroStream-related features when generating applications on [Micronaut Launch](https://micronaut.io/launch/), as well as on the Micronaut command-line interface (CLI):

- [MicroStream](https://micronaut.io/launch/?type=DEFAULT&name=demo&package=com.example&javaVersion=JDK_11&lang=JAVA&build=GRADLE&test=JUNIT&features=microstream&version=3.5.0) adds support for using MicroStream with Micronaut
- [MicroStream Cache](https://micronaut.io/launch/?type=DEFAULT&name=demo&package=com.example&javaVersion=JDK_11&lang=JAVA&build=GRADLE&test=JUNIT&features=microstream-cache&version=3.5.0) adds support for using MicroStream as a cache
- [MicroStream Rest](https://micronaut.io/launch/?type=DEFAULT&name=demo&package=com.example&javaVersion=JDK_11&lang=JAVA&build=GRADLE&test=JUNIT&features=microstream-rest&version=3.5.0) adds the MicroStream REST API to your Micronaut project for development

If you would like to learn more about using MicroStream as a high-performance persistence layer in a Micronaut framework application, you can start with [the documentation](https://micronaut-projects.github.io/micronaut-microstream/snapshot/guide/) and [this guide](https://guides.micronaut.io/latest/micronaut-microstream-persistence.html).

We are proud to have MicroStream as a member of our community of users. MicroStream is a great fit for Micronaut applications, and we’re looking forward to seeing all the great things community members build with these technologies. On behalf of the Micronaut Foundation, we thank MicroStream for their generous donation in support of the Micronaut framework.

The Micronaut Foundation is a not-for-profit organization that supports the Micronaut framework. The Foundation not only builds and supports an ecosystem of documentation, functionality, and services, it promotes and evangelizes the Framework as a leading technology in the JVM space. The generous support of our [Corporate Sponsors](https://micronaut.io/foundation/corporate-sponsorship/) allows the Foundation to continue to ensure technical innovation and advancement of the Framework as a free and open public-use software development toolkit for our growing global community.

To learn more about how you or your organization can support this Foundation, please [check us out](https://micronaut.io/foundation/)!
