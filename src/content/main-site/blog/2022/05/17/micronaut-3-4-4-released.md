The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.4.4](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.4.4)!

This release contains minor bug fixes and patch releases of several Micronaut Modules:

- [Micronaut Maven Plugin 3.2.4](https://github.com/micronaut-projects/micronaut-maven-plugin/releases/tag/v3.2.4)
- [Micronaut SQL 4.2.3](https://github.com/micronaut-projects/micronaut-sql/releases/tag/v4.2.3)
- [Micronaut JAX-RS 3.2.1](https://github.com/micronaut-projects/micronaut-jaxrs/releases/tag/v3.2.1)
- [Micronaut Oracle Cloud 2.1.3](https://github.com/micronaut-projects/micronaut-oracle-cloud/releases/tag/v2.1.3)
- [Micronaut MQTT 2.1.1](https://github.com/micronaut-projects/micronaut-mqtt/releases/tag/v2.1.1)
- [Micronaut OpenAPI 4.0.1](https://github.com/micronaut-projects/micronaut-openapi/releases/tag/v4.0.1)

Updating your Micronaut applications to 3.4.4 should be as easy as updating your Micronaut version in your build files.

Please feel free to [reach out to us if you need any assistance](https://objectcomputing.com/products/micronaut/consulting-support).

We hope that you enjoy [Micronaut framework 3.4!](https://micronaut.io/2022/03/22/micronaut-framework-3-4-0-released/)
