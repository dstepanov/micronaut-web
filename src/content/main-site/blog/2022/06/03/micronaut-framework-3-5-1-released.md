The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.5.1](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.5.1)!

This release contains minor bug fixes and a patch releases of [Micronaut Servlet](https://micronaut-projects.github.io/micronaut-servlet/latest/guide/).

If you have not yet updated to [Micronaut Framework 3.5.0](https://micronaut.io/2022/05/26/micronaut-framework-3-5-0/), this is a great opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support) if you need any assistance.
