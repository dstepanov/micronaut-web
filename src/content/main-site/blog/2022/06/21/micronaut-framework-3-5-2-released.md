The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.5.2](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.5.2)!

This release contains minor bug fixes and patch releases of [Micronaut Oracle Cloud](https://github.com/micronaut-projects/micronaut-oracle-cloud/releases/tag/v2.1.4), [Micronaut Email](https://github.com/micronaut-projects/micronaut-email/releases/tag/v1.2.3), and [Micronaut Spring](https://github.com/micronaut-projects/micronaut-spring/releases/tag/v4.1.1).

We have documented the [`ApplicationContextConfigurer` API](https://docs.micronaut.io/latest/api/io/micronaut/context/ApplicationContextConfigurer.html) which is the recommended approach to define a [default environment](https://docs.micronaut.io/latest/guide/#_default_environment).

If you have not yet updated to Micronaut framework 3.5.0, this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support) if you need any assistance.
