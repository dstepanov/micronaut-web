The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.4.1](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.4.1). It is a patch release.[The Micronaut framework follows semantic versioning](https://micronautpodcast.com/002.html). Thus, 3.4.1 contains bug fixes, module patch upgrades, and documentation improvements.

Moreover, if you use [Gradle](https://gradle.org/) to build your Micronaut applications, update the [Micronaut Gradle Plugins](https://plugins.gradle.org/plugin/io.micronaut.application) to 3.3.2.

Updating to 3.4.1 should be as easy as updating your Micronaut version in your build files. We hope that you enjoy [Micronaut Framework 3.4](https://micronaut.io/2022/03/22/micronaut-framework-3-4-0-released/).
