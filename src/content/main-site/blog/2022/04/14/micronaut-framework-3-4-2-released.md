The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.4.2](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.4.2)! The Micronaut framework follows [semantic versioning](https://micronautpodcast.com/002.html). Thus, this latest release is a patch release which contains bug fixes, module patch upgrades, and documentation improvements.

This release also contains patch releases of [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.2.3), [Micronaut Email](https://github.com/micronaut-projects/micronaut-email/releases/tag/v1.2.1), [Micronaut AOT](https://github.com/micronaut-projects/micronaut-aot/releases/tag/v1.0.3), and [Micronaut Micrometer](https://github.com/micronaut-projects/micronaut-micrometer/releases/tag/v4.2.1). Additionally, it upgrades to [Netty 4.1.76](https://netty.io/news/2022/04/12/4-1-76-Final.html).

Updating your Micronaut applications to 3.4.2 should be as easy as updating your Micronaut version in your build files.

If you need any assistance, please feel free to [reach out to us.](https://objectcomputing.com/products/micronaut/consulting-support)

We hope that you enjoy [Micronaut framework 3.4!](https://micronaut.io/2022/03/22/micronaut-framework-3-4-0-released/)
