The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.4.3](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.4.3)!

This release contains bug fixes around deserialization from Base64 into a byte array, the interception of Kotlin coroutine, validation group inheritance, and several documentation improvements.

Moreover, this release also contains a patch release of [Micronaut Problem-JSON](https://github.com/micronaut-projects/micronaut-problem-json/releases/tag/v2.2.3).

Updating your Micronaut applications to 3.4.3 should be as easy as updating your Micronaut version in your build files.

If you need any assistance, please feel free to [reach out to us.](https://objectcomputing.com/products/micronaut/consulting-support)

We hope that you enjoy [Micronaut framework 3.4!](https://micronaut.io/2022/03/22/micronaut-framework-3-4-0-released/)
