The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.7.4](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.7.4)!

This release contains bug fixes and documentation improvements. Moreover, it contains patch releases of several modules, including [Micronaut Security](https://github.com/micronaut-projects/micronaut-security/releases), [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.9.3), and [Micronaut Serialization](https://github.com/micronaut-projects/micronaut-serialization/releases/tag/v1.3.3).

If you have not yet updated to [Micronaut framework 3.7](https://micronaut.io/2022/09/21/micronaut-framework-3-7-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.5398010d44579cd3c326ff28c5086545.1660912926021.1664378508537.1664433503738.18&__hssc=207490764.2.1664433503738&__hsfp=2715702327) if you need any assistance.
