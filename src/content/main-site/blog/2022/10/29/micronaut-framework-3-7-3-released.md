The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.7.3](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.7.3)!

This release contains bug fixes and patch releases of several modules, including [Micronaut Test Resources](https://github.com/micronaut-projects/micronaut-test-resources/releases/tag/v1.1.3),[Servlet](https://github.com/micronaut-projects/micronaut-servlet/releases/tag/v3.3.2),[Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.8.1),[Kafka](https://github.com/micronaut-projects/micronaut-kafka/releases/tag/v4.4.1), and [Redis.](https://github.com/micronaut-projects/micronaut-redis/releases/tag/v5.3.1) Furthermore, it upgrades to SnakeYAML 1.33 and Netty 4.1.84.

Moreover, update your application to version 3.6.3 of the [Micronaut Gradle Plugins](https://plugins.gradle.org/search?term=micronaut) if you use Gradle.

If you have not yet updated to [Micronaut framework 3.7](https://micronaut.io/2022/09/21/micronaut-framework-3-7-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.5398010d44579cd3c326ff28c5086545.1660912926021.1664378508537.1664433503738.18&__hssc=207490764.2.1664433503738&__hsfp=2715702327) if you need any assistance.
