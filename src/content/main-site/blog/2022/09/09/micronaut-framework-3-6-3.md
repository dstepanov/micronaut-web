The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.6.3](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.6.3)!

This release contains bug fixes and patch releases of several modules, including [Micronaut Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.6.5), [Micronaut Email](https://github.com/micronaut-projects/micronaut-email/releases/tag/v1.3.2), [Micronaut Spring](https://github.com/micronaut-projects/micronaut-spring/releases/tag/v4.2.2), [Micronaut Tracing](https://github.com/micronaut-projects/micronaut-tracing/releases/tag/v4.2.2), and  [Micronaut Flyway.](https://github.com/micronaut-projects/micronaut-flyway/releases/tag/v5.4.1)

If you have not yet updated to [Micronaut framework 3.6](https://micronaut.io/2022/08/04/micronaut-framework-3-6-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.e7bbaa5fd0dc14f8605f277cc56ea4ec.1660849380803.1660849380803.1660906562333.2&__hssc=207490764.5.1660906562333&__hsfp=2447033234) if you need any assistance.
