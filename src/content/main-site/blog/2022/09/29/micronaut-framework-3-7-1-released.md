The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.7.1](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.7.1)!

This release contains bug fixes and patch releases of several modules, including [Micronaut CRaC](https://github.com/micronaut-projects/micronaut-crac/releases/tag/v1.0.1), [Micronaut SQL](https://github.com/micronaut-projects/micronaut-sql/releases/tag/v4.7.2), [Micronaut Test Resources](https://github.com/micronaut-projects/micronaut-test-resources/releases/tag/v1.1.2), and [Micronaut Problem+JSON `2.4.1`](https://github.com/micronaut-projects/micronaut-problem-json/releases/tag/v2.4.1).

Moreover, update your application to version 3.6.2 of the [Micronaut Gradle Plugins](https://plugins.gradle.org/search?term=micronaut) if you use Gradle.

If you have not yet updated to [Micronaut framework 3.7](https://micronaut.io/2022/09/21/micronaut-framework-3-7-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.5398010d44579cd3c326ff28c5086545.1660912926021.1664378508537.1664433503738.18&__hssc=207490764.2.1664433503738&__hsfp=2715702327) if you need any assistance.

## MICRONAUT FRAMEWORK 3.6.4 RELEASED

Moreover, we are excited to announce the release of [Micronaut framework 3.6.4](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.6.4). It contains bug fixes and patch releases of several modules, including [Micronaut SQL](https://github.com/micronaut-projects/micronaut-sql/releases/tag/v4.6.6) and [Micronaut Hibernate Validator](https://github.com/micronaut-projects/micronaut-hibernate-validator/releases/tag/v3.0.2).
