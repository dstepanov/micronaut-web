The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.6.2](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.6.2)!

This release contains bug fixes and patch releases of several modules, including [Micronaut Flyway](https://github.com/micronaut-projects/micronaut-flyway/releases/tag/v5.4.1), [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.7.1), [Micronaut Serialization](https://github.com/micronaut-projects/micronaut-serialization/releases/tag/v1.3.2), [Micronaut Data, and](https://github.com/micronaut-projects/micronaut-data/releases/tag/v3.7.3) [Micronaut Problem-JSON.](https://github.com/micronaut-projects/micronaut-problem-json/releases/tag/v2.4.1)

Moreover, update your application to version 3.5.3 of the [Micronaut Gradle Plugins](https://plugins.gradle.org/search?term=micronaut) if you use Gradle.

**This patch also updates to [SnakeYAML 1.31](https://bitbucket.org/snakeyaml/snakeyaml/wiki/Changes). This version of SnakeYAML addresses [CVE-2022-25857.](https://nvd.nist.gov/vuln/detail/CVE-2022-25857)**

If you have not yet updated to [Micronaut framework 3.6](https://micronaut.io/2022/08/04/micronaut-framework-3-6-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.e7bbaa5fd0dc14f8605f277cc56ea4ec.1660849380803.1660849380803.1660906562333.2&__hssc=207490764.5.1660906562333&__hsfp=2447033234) if you need any assistance.

## Micronaut Framework 3.5.5 Released

Moreover, we are excited to announce also the release of [Micronaut framework 3.5.5](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.5.5). It upgrades also to [SnakeYAML 1.31](https://bitbucket.org/snakeyaml/snakeyaml/wiki/Changes).  It contains bug fixes and patch releases of several modules, including [Micronaut Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.6.4), and [Micronaut AOT.](https://github.com/micronaut-projects/micronaut-aot/releases/tag/v1.1.1)
