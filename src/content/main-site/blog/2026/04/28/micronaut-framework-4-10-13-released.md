The Micronaut Foundation is excited to announce the release of Micronaut framework 4.10.13! It contains releases of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.10.22),[Micronaut Sourcegen](https://github.com/micronaut-projects/micronaut-sourcegen/releases/tag/v1.8.5), and [Micronaut SQL](https://github.com/micronaut-projects/micronaut-sql/releases/tag/v6.3.4).

The latest version of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.10.22), included in this release, contains fixes for two security advisories related to unbounded caches [in TimeConverterRegistrar](https://github.com/micronaut-projects/micronaut-core/security/advisories/GHSA-8hjv-92q9-g4xj) and [ResourceBundleMessageSource.](https://github.com/micronaut-projects/micronaut-core/security/advisories/GHSA-3rfq-4wpf-qqw3)

**We strongly recommend that users update to Micronaut 4.10.13.**If you are still using Micronaut 3, update to [Micronaut 3.10.6](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.10.6), which includes the necessary patches.

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
