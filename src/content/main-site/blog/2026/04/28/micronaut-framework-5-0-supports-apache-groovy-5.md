Micronaut Framework 5.0, to be released in Q2 2026, supports [Apache Groovy 5](https://groovy-lang.org/releasenotes/groovy-5.0.html). Apache Groovy 5.0 adds new features such as:

- Broader JDK Support
- Substantial Extension Method Expansion
- Enhanced AST Transforms
- Revamped REPL (groovysh)
- Improved Java Interoperability
- Modern Web Standards
- Type Checking Enhancements
- Additional Language Goodies
