The Micronaut Foundation is excited to announce the release of Micronaut framework 4.10.10! It contains releases of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.10.18), and [Micronaut Test.](https://github.com/micronaut-projects/micronaut-test/releases/tag/v4.10.3)

## Potential denial of service attacks fixes

Today, we published two security advisories that are fixed in Micronaut Core 4.10.10. Thus, strongly recommend that every user update to the latest version of Micronaut.

- [DoS via crafted form-urlencoded body binding with descending array indices](https://github.com/micronaut-projects/micronaut-core/security/advisories/GHSA-43w5-mmxv-cpvh)
- [Denial of Service in HTML error response caching](https://github.com/micronaut-projects/micronaut-core/security/advisories/GHSA-2hcp-gjrf-7fhc)

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
