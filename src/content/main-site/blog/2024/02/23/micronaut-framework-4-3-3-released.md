The Micronaut Foundation is excited to announce the release of Micronaut framework 4.3.3!

It contains patch releases of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.3.8), [Micronaut Hibernate Validator](https://github.com/micronaut-projects/micronaut-hibernate-validator/releases/tag/v4.2.1), [Micronaut Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v4.6.2), [Micronaut GCP](https://github.com/micronaut-projects/micronaut-gcp/releases/tag/v5.4.1), and [Micronaut Test](https://github.com/micronaut-projects/micronaut-test/releases/tag/v4.2.1).

Moreover, update your application to version 4.3.3 of the [Micronaut Gradle Plugins](https://plugins.gradle.org/u/micronaut) if you use Gradle.

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
