The Micronaut Foundation is excited to announce the release of Micronaut framework 4.0.6!

It contains patch releases of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/v4.0.6) and it upgrades [Micronaut Spring](https://github.com/micronaut-projects/micronaut-spring/releases/tag/v5.0.2), [Micronaut JAX-RS](https://github.com/micronaut-projects/micronaut-jaxrs/releases/tag/v4.0.2), [Micronaut Servlet](https://github.com/micronaut-projects/micronaut-servlet/releases/tag/v4.0.5), [Micronaut Validation](https://github.com/micronaut-projects/micronaut-validation/releases/tag/v4.0.2), [Micronaut Redis](https://github.com/micronaut-projects/micronaut-redis/releases/tag/v6.0.2), [Micronaut Tracing](https://github.com/micronaut-projects/micronaut-tracing/releases/tag/v5.0.2), [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v4.0.3), and [Micronaut Kafka](https://github.com/micronaut-projects/micronaut-kafka/releases/tag/v5.0.3) modules to patch releases.

Moreover, update your application to version 4.0.3 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/u/micronaut)

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
