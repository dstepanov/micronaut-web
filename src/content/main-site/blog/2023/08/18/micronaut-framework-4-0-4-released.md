The Micronaut Foundation is excited to announce the release of Micronaut framework 4.0.4!

Micronaut framework 4.0.4 upgrades [Micronaut GraphQL](https://github.com/micronaut-projects/micronaut-graphql/releases/tag/v4.0.2), [Micronaut Neo4J](https://github.com/micronaut-projects/micronaut-neo4j/releases/tag/v6.0.2), [Reactor](https://github.com/micronaut-projects/micronaut-reactor/releases/tag/v3.0.2), [Servlet](https://github.com/micronaut-projects/micronaut-servlet/releases/tag/v4.0.2),[Cache](https://github.com/micronaut-projects/micronaut-cache/releases/tag/v4.0.2), and [CRaC](https://github.com/micronaut-projects/micronaut-crac/releases/tag/v2.0.3) modules to patch releases.

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.

## Micronaut Framework 3 Releases

We are happy to announce Micronaut Framework releases [3.10.1](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.10.1) and [3.9.6](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.9.6) – patch releases that contain bug fixes.
