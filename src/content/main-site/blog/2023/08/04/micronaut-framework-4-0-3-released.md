The Micronaut Foundation is excited to announce the release of Micronaut framework 4.0.3!

Micronaut framework 4.0.3 is a patch release that contains bug fixes and upgrades.

It contains a patch release of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.0.3), and it upgrades [Maven Plugin](https://github.com/micronaut-projects/micronaut-maven-plugin/releases/tag/v4.0.4), [Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v4.0.3),[Multi-tenancy](https://github.com/micronaut-projects/micronaut-multitenancy/releases/tag/v5.0.2), [GCP](https://github.com/micronaut-projects/micronaut-gcp/releases/tag/v5.0.3), [RabbitMQ](https://github.com/micronaut-projects/micronaut-rabbitmq/releases/tag/v4.0.2), and [Securit](https://github.com/micronaut-projects/micronaut-security/releases/tag/v4.0.3) y modules to patch releases.

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
