The Micronaut Foundation is excited to announce the release of Micronaut framework 4.1.2!

It contains a patch release of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/v4.1.6) and upgrades [Micronaut Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v4.1.2) to a patch release.

## NEXT STEPS

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
