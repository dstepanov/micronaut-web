The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.4!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.4)

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.4 includes a patch release of several modules – [Micronaut OpenAPI](https://github.com/micronaut-projects/micronaut-openapi/releases/tag/v4.8.3), [Micronaut Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v3.9.6), [Micronaut RabbitMQ](https://github.com/micronaut-projects/micronaut-rabbitmq/releases/tag/v3.4.1), [Micronaut Azure](https://github.com/micronaut-projects/micronaut-azure/releases/tag/v3.7.1), [Micronaut Micrometer](https://github.com/micronaut-projects/micronaut-micrometer/releases/tag/v4.7.2), and [Micronaut Test](https://github.com/micronaut-projects/micronaut-test/releases/tag/v3.8.2).

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
