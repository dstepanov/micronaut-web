The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.6!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.6)

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.6 includes patch releases of several modules – [Micronaut Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.9.3), and [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.10.9).

Moreover, update your application to version 3.7.3 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/u/micronaut)

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
