The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.8!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.8)

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.8 upgrades to [Netty 4.1.90](https://netty.io/news/2023/03/14/4-1-90-Final.html) and patch releases of several modules – [Micronaut Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v3.9.7), [Micronaut Views](https://github.com/micronaut-projects/micronaut-views/releases/tag/v3.8.2), [Micronaut OpenAPI](https://github.com/micronaut-projects/micronaut-openapi/releases/tag/v4.8.6), [Micronaut Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.9.4), and [Micronaut Maven Plugin](https://github.com/micronaut-projects/micronaut-maven-plugin/releases/tag/v3.5.3).

Moreover, update your application to version 3.7.7 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/u/micronaut)

## Next Steps

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
