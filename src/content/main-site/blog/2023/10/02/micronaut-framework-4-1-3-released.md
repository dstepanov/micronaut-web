The Micronaut Foundation is excited to announce the release of Micronaut framework 4.1.3!.

It contains a patch release of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/v4.1.8) and upgrades [Micronaut Oracle Cloud](https://github.com/micronaut-projects/micronaut-oracle-cloud/releases/tag/v3.0.6), [Micronaut Serialization](https://github.com/micronaut-projects/micronaut-serialization/releases/tag/v2.2.5), and [Micronaut SQL](https://github.com/micronaut-projects/micronaut-sql/releases/tag/v5.0.2) to patch releases.

Moreover, it updates to [Netty 4.1.99](https://netty.io/news/2023/09/21/4-1-99-Final.html).

Moreover, update your application to version 4.0.4 of the [Micronaut Gradle Plugins. if you use Gradle.](https://plugins.gradle.org/u/micronaut)

## Next Steps

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
