The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.2!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.2)

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.2 includes patch releases of several modules – [Micronaut Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.9.2), [Micronaut Views](https://github.com/micronaut-projects/micronaut-views/releases/tag/v3.8.1), [Micronaut Micrometer](https://github.com/micronaut-projects/micronaut-micrometer/releases/tag/v4.7.1), and [Micronaut Servlet](https://github.com/micronaut-projects/micronaut-servlet/releases/tag/v3.3.5).

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
