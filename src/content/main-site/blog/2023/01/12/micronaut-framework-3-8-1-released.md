The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.1!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.1)

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.1 includes patch releases of several modules –[Micronaut Servlet](https://github.com/micronaut-projects/micronaut-servlet/releases/tag/v3.3.3),[Micronaut Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v3.9.4), and [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.10.5).

Moreover, update your application to version 3.7.0 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/u/micronaut)

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://objectcomputing.com/products/micronaut/consulting-support?__hstc=207490764.5398010d44579cd3c326ff28c5086545.1660912926021.1664378508537.1664433503738.18&__hssc=207490764.2.1664433503738&__hsfp=2715702327) if you need any assistance.
