The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.3!](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.3)

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.3 includes a patch release of several modules – [Micronaut OpenAPI](https://github.com/micronaut-projects/micronaut-openapi/releases/tag/v4.8.2) and [Micronaut Oracle Cloud](https://github.com/micronaut-projects/micronaut-oracle-cloud/releases/tag/v2.3.2).

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
