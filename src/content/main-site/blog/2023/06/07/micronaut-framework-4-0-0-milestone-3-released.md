The Micronaut Foundation is excited to announce the release of Micronaut framework 4 Milestone 3. We have published several blog posts about the [changes introduced in Micronaut framework 4.0](https://micronaut.io/tag/micronaut4/). You can also find more details in the [What’s New](https://docs.micronaut.io/4.0.0-M2/guide/index.html#whatsNew) and [breaking changes](https://docs.micronaut.io/4.0.0-M2/guide/index.html#_core_changes) sections of the Micronaut framework 4.0 core documentation.

This milestone release includes many new features. Now, you can use the [evaluated expressions](https://docs.micronaut.io/4.0.0-M2/guide/#evaluatedExpressions) with the [@Secure annotation](https://micronaut-projects.github.io/micronaut-security/snapshot/guide/#securedExpression). Additionally, you can develop Lambda functions integration with the new [Micronaut AWS Lambda integration](https://micronaut-projects.github.io/micronaut-aws/snapshot/guide/#lambda), which uses the Micronaut AWS Lambda Java Events.

## Generating a Micronaut framework 4 application

You can generate a Micronaut framework 4.0.0-M3 application with [Micronaut Launch](https://launch.micronaut.io/) or in the Micronaut CLI. The easiest way to install 4.0.0-M3 CLI is with [sdkman](https://sdkman.io/). To do so, type sdk install micronaut 4.0.0-M3, and then run mn-create.

## upgrading to Micronaut framework 4

Interested in trying Micronaut framework 4.0.0-M3? We have published an [upgrade to Micronaut framework 4 guide](https://micronaut.io/2023/05/09/upgrade-to-micronaut-framework-4-0-0/) to help you.

Thanks to all those who contributed to this release!

We look forward to your feedback as we progress toward a GA (General Availability) release.
