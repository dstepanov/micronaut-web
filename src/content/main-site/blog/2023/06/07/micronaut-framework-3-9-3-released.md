The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.9.3](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.9.3)!

This is a patch release which contains bug fixes and upgrades.

Micronaut 3.9.3 upgrades to [Netty 4.1.92](https://netty.io/news/2023/04/25/4-1-92-Final.html) with additional upgrades of [Servlet](https://micronaut-projects.github.io/micronaut-servlet/latest/guide/) and [AWS](https://micronaut-projects.github.io/micronaut-aws/latest/guide/) modules to patch releases.

If you use Gradle, we suggest updating your application to version 3.7.10 of the [Micronaut Gradle Plugins.](https://plugins.gradle.org/u/micronaut)

## Next steps

If you haven’t yet updated to Micronaut framework 3.9, this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
