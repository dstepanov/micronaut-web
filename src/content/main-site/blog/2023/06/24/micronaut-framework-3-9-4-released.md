The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.9.4](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.9.4)!

This is a patch release which contains bug fixes and upgrades.

Micronaut Framework 3.9.4 upgrades to [Netty 4.1.94](https://netty.io/news/2023/06/19/4-1-94-Final.html) with additional upgrades of [Security](https://github.com/micronaut-projects/micronaut-security/releases/tag/v3.11.1) and [Servlet](https://github.com/micronaut-projects/micronaut-servlet/releases/tag/v3.3.8) modules to patch releases.

## Next steps

If you haven’t yet updated to Micronaut framework 3.9, this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
