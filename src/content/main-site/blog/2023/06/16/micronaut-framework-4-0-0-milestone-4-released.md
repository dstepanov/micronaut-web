The Micronaut Foundation is excited to announce the release of Micronaut framework 4 Milestone 4. We have published several blog posts about the [changes introduced in Micronaut framework 4.0](https://micronaut.io/tag/micronaut4/). You can also find more details in the [What’s New](https://docs.micronaut.io/4.0.0-M2/guide/index.html#whatsNew) and [breaking changes](https://docs.micronaut.io/4.0.0-M2/guide/index.html#_core_changes) sections of the Micronaut framework 4.0 core documentation.

## Generating a Micronaut Framework 4 Application

You can generate a Micronaut framework 4.0.0-M4 application with [Micronaut Launch](https://launch.micronaut.io/) or in the Micronaut CLI. The easiest way to install 4.0.0-M4 CLI is with [sdkman](https://sdkman.io/). To do so, type sdk install micronaut 4.0.0-M4, and then run mn-create.

## Upgrading to Micronaut Framework 4

Interested in trying Micronaut framework 4.0.0-M4? We have published an [upgrade to Micronaut framework 4 guide](https://micronaut.io/2023/05/09/upgrade-to-micronaut-framework-4-0-0/) to help you.

Thanks to all those who contributed to this release!

We look forward to your feedback as we progress toward a GA (General Availability) release.
