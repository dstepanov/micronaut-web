The Micronaut Foundation is excited to announce the release of Micronaut framework 4.0.1!

Micronaut framework 4.0.1 is a patch release that contains bug fixes and upgrades.

It contains a patch release of [Micronaut Core](https://github.com/micronaut-projects/micronaut-core/releases/tag/v4.0.1), and it upgrades [Data](https://github.com/micronaut-projects/micronaut-data/releases/tag/v4.0.2), [Problem+JSON](https://github.com/micronaut-projects/micronaut-problem-json/releases/tag/v3.0.2), [Hibernate Validator](https://github.com/micronaut-projects/micronaut-hibernate-validator/releases/tag/v4.0.2), and [Email](https://github.com/micronaut-projects/micronaut-email/releases/tag/v2.0.2) modules to patch releases.

Moreover, update your application to version 4.0.1 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/u/micronaut)

## Next Steps

If you haven’t yet updated to [Micronaut framework 4](https://micronaut.io/2023/07/14/micronaut-framework-4-0-0-released/), this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
