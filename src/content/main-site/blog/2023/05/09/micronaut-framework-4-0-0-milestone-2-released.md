The Micronaut Foundation is excited to announce the release of Micronaut framework 4 Milestone 2. This is the first public milestone release and includes many new features.

We have published [several blog posts about the changes introduced in Micronaut framework 4.0](https://micronaut.io/tag/micronaut4/). You can find more details in the [*What’s new*](https://docs.micronaut.io/4.0.0-M2/guide/index.html#whatsNew) and [* breaking changes*](https://docs.micronaut.io/4.0.0-M2/guide/index.html#_core_changes) sections of the Micronaut framework 4.0 core documentation.

## Generating a Micronaut Framework 4 application

You can generate a Micronaut framework 4.0.0-M2 application with [Micronaut Launch](https://launch.micronaut.io/) or with the Micronaut CLI. The easiest way to install 4.0.0-M2 CLI is with [sdkman](https://sdkman.io/). Type `sdk install micronaut 4.0.0-M2`, and then run `mn-create`.

## Upgrading to Micronaut Framework 4

To help you try Micronaut framework 4.0.0-M2, we have published the following [upgrade to Micronaut Framework 4 guide](https://micronaut.io/2023/05/09/upgrade-to-micronaut-framework-4-0-0/).

Thanks to all those who contributed to the release, and we look forward to your feedback as we progress towards a GA (General Availability) release.
