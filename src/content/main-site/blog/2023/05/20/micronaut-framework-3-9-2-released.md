The Micronaut Foundation is excited to announce the release of [Micronaut Framework 3.9.2](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.9.2)!

This is a patch release, and it contains bug fixes. Additionally, Micronaut 3.9.2 upgrades to patch releases of several modules –[Azure](https://micronaut-projects.github.io/micronaut-azure/latest/guide/),[AWS](https://micronaut-projects.github.io/micronaut-aws/latest/guide/),[GCP](https://micronaut-projects.github.io/micronaut-gcp/latest/guide/),[OpenAPI](https://micronaut-projects.github.io/micronaut-openapi/latest/guide/),[SQL](https://micronaut-projects.github.io/micronaut-sql/latest/guide/), and [Kubernetes](https://micronaut-projects.github.io/micronaut-kubernetes/latest/guide/).

## Next steps

If you still need to update to Micronaut framework 3.9, this is an excellent opportunity to do so!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
