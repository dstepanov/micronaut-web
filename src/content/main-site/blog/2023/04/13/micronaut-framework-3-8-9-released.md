The Micronaut Foundation is excited to announce the release of [Micronaut framework 3.8.9](https://github.com/micronaut-projects/micronaut-core/releases/tag/v3.8.9)!

This is a patch release, and it contains bug fixes. Moreover, Micronaut 3.8.9 upgrades to [Netty 4.1.91](https://netty.io/news/2023/04/03/4-1-91-Final.html) and patch releases of several modules – [Micronaut AWS](https://github.com/micronaut-projects/micronaut-aws/releases/tag/v3.10.10), and [Micronaut Kafka](https://github.com/micronaut-projects/micronaut-kafka/releases/tag/v4.5.3).

Moreover, update your application to version 3.7.8 of the [Micronaut Gradle Plugins if you use Gradle.](https://plugins.gradle.org/u/micronaut)

## Next Steps

If you still need to update to [Micronaut framework 3.8](https://micronaut.io/2022/12/27/micronaut-framework-3-8-0-released/), this is an excellent opportunity to do it!

Please feel free to [reach out to us](https://micronaut.io/support/) if you need any assistance.
