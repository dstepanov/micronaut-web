= Snippet Gallery Fixture

This fixture exercises generated snippet rendering across source blocks, snippet macros, dependencies, configuration samples, and configuration property tables.

== Ordinary Source Blocks

----
$ curl http://localhost:8080/hello
Hello World
----

[source,bash]
----
./gradlew test

.Java source with callouts
[source,java]
----
class HelloController { // <1>
}
----
<1> The controller source callout is rendered inside the card footer.

.Properties source with standalone callout marker
[source,properties]
----

micronaut.server.port=8080
----
